아기용 DVD 플레이어 가이드
최초 생성 : 16-03-24 07:19:03 // 최종 수정 : 16-03-29 13:58:53

이 글은 2016년 03월 02일 15시 36분 에 마지막으로 수정되었습니다.

이 글의 시각

  • 이 글은 일부 광고를 포함할 수 있습니다.
  • 다만 광고는 컨텐츠 내용에 어떠한 영향도 미치지 않습니다.
  • 나쁜데 광고라고 추천하거나 이런거 없습니다.
  • 후기 섹션의 경우 홍보 포스팅은 인용하지 않습니다.
  • 인비오한테 한푼도 안받았습니다.

몹시 바쁜 사람들을 위한 요약

  • 일반 사용자 : 인비오 PD-7600 사면 됩니다.
  • 카세트 테이프 기능 필요한 사람 : 인비오 PD-8400사면 됩니다.
  • 휴대용으로 화면하고 같이 있는 게 필요한 사람 : 인비오 PD-2300 사면 됩니다.

아기 dvd 플레이어

아기들 dvd 컨텐츠를 보거나 들을 수 있는 플레이어.
사실 특별히 아기 dvd 플레이어라는 시장은 존재하지 않는다.
그냥 일반 dvd 플레이어 중에서 아기들용으로 쓰기 좋은 DVD 플레이어 목록이다.

왜 필요한가?

아기가 2-3세를 넘어가면 아기들용 멀티미디어 컨텐츠가 대부분 다 DVD 형태로 나온다.
이건 베이비페어(베페) 같은 데 가보면 확실히 알 수 있다.

듣기기능 혹은 보면서 듣는 기능

어른들이 생각할 때는 기능이 하나라도 더 있는 것이 좋아보이지만, 아기들 입장에서는 별로 그렇지 않다.
사람은 소리만 들을 때와 화면으로 함께 볼 때 받아들이는 것이 전혀 다르다.
소리만 들으면 아이의 상상력을 자극하고 생각을 하게 만들지만, 화면을 같이 보게 되면 단순히 인지자극으로 받아들이기만 한다.
어른들의 사정으로 아이의 시선을 빼앗아야 할 때 외에는 좋은 선택은 아니다.
TV도 잘 안보여주려고 하는 이유도 마찬가지다.

그래서 가능하면 듣기 기능과 화면 기능이 분리된 DVD플레이어를 추천한다.
관련글 : 아기한테 보여줄 DVD 플레이어 추천부탁드려요^^

플레이어 목록

인비오

네이버에 검색 해보니 대부분 인비오 제품을 쓴다.
돈받고 쓴 리뷰는 아닌 것 같아서 먼저 인비오에 대해 찾아본다.

PD-7600

기능

CD, DVD, USB를 통한 MP3, 라디오 기능이 포함된 DVD 플레이어.
카세트 테이프 기능은 없다.
요새는 카세트 테이프로 새로운 컨텐츠가 나오진 않으니 큰 상관은 없을 것 같다.

블로그에서 리뷰가 제일 많이 나오는 상품. 가성비가 좋은 것 같다.
최저가는 2016년 2월 18일 기준 지마켓에서 80,400원 이다.

후기

너무 많아서 네이버 검색으로 대신함

PD-8400

기능

cd, dvd, 라디오, USB를 통한 MP3, 카세트

블로그에서 2번째로 많이 나오는 제품. 카세트 테이프가 있으므로 혹시 쓸 일이 있으면 이걸 사용하면 된다.
최저가는 2016년 2월 18일 기준 11번가 82,020 원

뽑기운이 좀 있다고 한다.
